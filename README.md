# CopyStack

A mobile-first speed puzzle PWA inspired by the vintage Jumbo board game "Number Rumba".

A target card shows an arrangement of colored blocks. Rearrange the blocks on the board to copy the card exactly — as fast as possible, in as few moves as possible. One extra dashed **buffer column** is available for temporary moves; it must be empty in the solution.

## How to play

1. Pick a difficulty (Easy / Medium / Hard / Insane). Hard and Insane add numbers: both color **and** number must match.
2. Press **Start** — the card flips face up and the timer starts.
3. Tap a column to pick up its top block, tap another column to drop it. Tap again to deselect.
4. Match the card. Score per card: `round(multiplier × 200000 / (seconds + 5 × moves)^1.3)` — moves cost more than seconds, slow runs collapse fast, and difficulty multipliers (Easy ×1, Medium ×4, Hard ×6, Insane ×12) make the harder levels worth far more.

High scores per difficulty are saved locally. 

## Develop

```sh
npm install
npm run dev      # dev server
npm test         # unit tests (vitest)
```

## Build & deploy

```sh
npm run build    # static bundle in dist/
npm run preview  # serve the production build locally
```

`dist/` is fully static — deploy to Netlify, Vercel, GitHub Pages, or any static host. The service worker caches the app on first load, so it works offline and is installable to the home screen on iOS/Android.

Icons are generated by `node scripts/make-icons.mjs` (already committed in `public/icons/`).
